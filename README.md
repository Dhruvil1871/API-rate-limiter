# API Rate Limiter & Abuse Protection Service

A configurable API Rate Limiting and Abuse Protection Service built using Django, Redis, and PostgreSQL.

## Features

* Fixed Window Rate Limiting
* Sliding Window Rate Limiting
* Token Bucket Rate Limiting
* Redis-backed request tracking
* Endpoint-specific policies
* IP-based rate limiting
* Client-specific rate limiting
* Request logging
* Analytics dashboard
* Rule management
* Automated testing

## Tech Stack

* Python
* Django
* Redis
* PostgreSQL

Currently under active development.
